# gatsby-plugin-antd
Use Ant Design with Gatsby. This plugin uses the babel-plugin-import to dynamically import Ant Design components.

## Install
`npm install gatsby-plugin-antd --save`

## How to use
1. Include the plugin in your `gatsby-config.js` file.

```javascript
// in gatsby-config.js
plugins: [
  'gatsby-plugin-antd'
]
```
